# Looking Through Write-up

We are presented with an image of well...nevermind. But the challenge description wants us to find "a very famous trade in late 2025". 

We check the metadata of the image to find among useless stuff, a simple set of coordinates:
```text
Transmission Reference          : Job_ID_40.7737N73.9654W_DO_NOT_DELETE
```
Coordinates: `(40.7737N, 73.9654W)`

Next, just using simple google searches, we find the trade being referred to and the flag as `TACHYON{Gustav_Klimt}`.

**Flag:** `TACHYON{Gustav_Klimt}`
