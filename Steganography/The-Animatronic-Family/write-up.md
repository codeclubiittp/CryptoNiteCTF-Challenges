# The Animatronic Family Write-up

The given recording is a **Martin1** encoded SSTV signal. 

Putting this in a tool like `qsstv` gives one a PNG image with the flag which is encrypted by base64 and ROT-13 (in that order).

So using a tool like CyberChef one can find the flag.

**Flag:** `TACHYON{D0ubl3_ch3cked_qu3stion_13d3}`
