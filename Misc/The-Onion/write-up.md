# The Onion Write-up

This challenge consisted of a folder which was `gunzip`ped -> `tar`red -> `zip`ped a 100 times. 

The way to do this challenge was to write a Python script (or any other script) which extracted the folders accordingly to get the flag.
A sample script is in `sample_script.py`.

Once you get the base64 flag, decode it to get `TACHYON{1_w0nd3r_wh0s_b3hind_th3_m4sk_3hf84hfr}`.

**Flag:** `TACHYON{1_w0nd3r_wh0s_b3hind_th3_m4sk_3hf84hfr}`
